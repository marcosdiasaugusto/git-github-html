Na linha de comando, primeiro verifique se você instalou o Git:

Em todos os sistemas operacionais:

git --version

Em sistemas operacionais semelhantes ao UNIX:


which git

Se nada for retornado, ou o comando não for reconhecido, você pode ter que instalar o Git em seu sistema
baixando e executando o instalador. Veja a página inicial do Git para uma instalação excepcionalmente clara e fácil
instruções.
Depois de instalar o Git, configure seu nome de usuário e endereço de e-mail. Faça isso antes de fazer um commit.
