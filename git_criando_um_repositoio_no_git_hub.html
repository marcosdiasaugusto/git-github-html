Criar um novo repositório no Github (ex. geek-git1)

Novo Repositório

No VSC File > Open Folder (mesmo nome criado no github)
Criar novo arquivo README.md

Nova Pasta

Seguir a sequencia de comando abaixo


echo "# geek-git1" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/marcosdiasaugusto/geek-git1.git
git push -u origin main
