Clone um repositório

O comando git clone é usado para copiar um repositório Git existente de um servidor para a máquina local.

Por exemplo, para clonar um projeto GitHub:

cd 

git clone https://github.com/username/projectname.git

Para clonar um projeto BitBucket:

cd 

git clone https: //yourusername@bitbucket.org/username/projectname.git

Isso cria um diretório chamado projectname na máquina local, contendo todos os arquivos no Git remoto
repositório. Isso inclui arquivos de origem para o projeto, bem como um subdiretório .git que contém todo o
histórico e configuração do projeto.

Para especificar um nome diferente do diretório, por exemplo, Minha pasta:

git clone https://github.com/username/projectname.git MyFolder

Ou para clonar no diretório atual:

git clone https://github.com/username/projectname.git.

Observação:

1. Ao clonar para um diretório especificado, o diretório deve estar vazio ou inexistente.
2. Você também pode usar a versão ssh do comando:

git clone git@github.com: username / projectname.git

A versão https e a versão ssh são equivalentes. No entanto, alguns serviços de hospedagem, como o GitHub, recomendam
que você usa https em vez de ssh.
