O git é descentralizado e cada repositório no git é um nó na rede. Ao trabalhar com 
médios e grandes projetos definimos um repositório como uma central onde cada desenvolvedor 
com seu repositório local submete os códigos/dados.
Exemplo de repositório Bare é o github. Teremos um repositório Bare Repository e dois 
desenvolvedores vao submeter os dados para esse repositório central.
No VSC > Open Folder > Secao_04_Bare

Open Folder > Julius

Agora o git push não deu erro por ser um bare repository
Git push envia seus commits para o repositório central remoto.
